# Epstein Africa

Public interest journalism database documenting Jeffrey Epstein's connections to the African continent, built from documents released under the DOJ Epstein Files Transparency Act.

**https://www.epsteinafrica.com**

## What this is

A searchable archive of emails, stories, and person profiles drawn from 1.78 million documents released by the U.S. Department of Justice. Every claim is anchored to a specific document ID verifiable against [jmail.world](https://jmail.world).

## Features

- Full-text search across 2,300+ emails (FTS5)
- 33 sourced investigative stories
- 38 person profiles with email correspondence
- Interactive network graph (D3)
- Country filtering (30+ countries)
- Data export (CSV, JSON)
- RSS feed
- Dynamic Open Graph images

## Stack

Next.js 14 (Pages Router) / SQLite / Vercel / Cloudflare
